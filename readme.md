# ⏰ Digital Clock & Stopwatch

A modern and responsive **Digital Clock & Stopwatch** built using **HTML, CSS, and JavaScript**. This project features a real-time digital clock, a high-precision stopwatch, and lap time recording with a clean glassmorphism user interface—without using any images.

---


## 🚀 Features

* 🕒 Live Digital Clock
* 📅 Current Date and Day
* 🌗 12-Hour Time Format (AM/PM)
* ▶️ Start Stopwatch
* ⏸️ Pause Stopwatch
* 🔄 Reset Stopwatch
* 🚩 Record Unlimited Lap Times
* 📱 Fully Responsive Layout

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript 
* Google Fonts (Poppins)
* Font Awesome
